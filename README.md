# KM Intelligence

Aplicación simple en Streamlit con contenido académico y análisis de datos.

## Qué incluye

- Página de inicio del proyecto.
- Artículo sobre web scraping (conceptos, marco legal y buenas prácticas).
- Estudio sectorial de precios de vehículos en República Dominicana.

## Cómo usarlo

1. Instala las dependencias del proyecto.
2. Crea tu archivo `.env` con las credenciales de base de datos (si usarás el estudio de vehículos).
3. Ejecuta la app:

```bash
streamlit run home.py
```

## Nota

Este proyecto está pensado para consulta y análisis interactivo.
